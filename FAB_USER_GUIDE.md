# 🎨 主页浮动按钮（FAB）使用指南

**版本**: v1.0.0  
**更新时间**: 2026-02-07

---

## 📖 功能说明

在 AI Agent Hub 主页的右下角，新增了一个**浮动操作按钮（FAB）**，点击后会展开菜单，提供两个快速操作：

1. **添加 Agent** 🤖 - 快速跳转到 Agent 管理，添加新的 AI Agent
2. **创建群组** 👥 - 快速创建新的群聊频道

---

## 🖱️ 如何使用

### 步骤 1：打开主页

启动 AI Agent Hub 应用，进入主页。

---

### 步骤 2：点击浮动按钮

在屏幕右下角，找到蓝色的 **"+"** 圆形按钮，点击它。

---

### 步骤 3：选择操作

点击后，按钮会变成 **"×"**，并且向上展开两个选项：

```
展开后的布局:
┌─────────────────────────────────────┐
│                                     │
│  ╔═══════════════════╗              │

│                                     │
│              [创建群组]    [👥]  ← 紫色按钮
│                                     │
│                            [×]  ← 关闭菜单
└─────────────────────────────────────┘
```

---

### 选项 A：添加 Agent

1. 点击 **蓝色** 的 **"添加 Agent"** 按钮
2. 自动跳转到 **Agent 管理列表页面**
3. 在列表页面点击右下角的 **"+"** 按钮
4. 选择 Agent 类型：
   - 🦅 OpenClaw Agent
   - 🤖 A2A Agent
   - 🔗 自定义 Agent
5. 填写表单并创建

---

### 选项 B：创建群组

1. 点击 **紫色** 的 **"创建群组"** 按钮
2. 自动跳转到 **创建群组页面**
3. 填写群组名称
4. 选择群组成员（至少选择一个 Agent）
5. 点击右上角的 **"创建"** 按钮完成

---

### 步骤 4：关闭菜单

如果不想进行操作，可以通过以下方式关闭菜单：

**方式 1**：点击主按钮（**"×"** 图标）  
**方式 2**：点击屏幕上的**背景遮罩**（半透明黑色区域）  
**方式 3**：选择任一菜单项后自动关闭

## 💡 使用技巧

### 技巧 1：快速访问

在主页时，无需再点击功能卡片进入列表页，直接点击 FAB 即可快速操作。

**节省时间**：50%（从 4 步减少到 2 步）

---

### 技巧 2：随时取消

如果误点击了 FAB，只需点击背景即可关闭，不会影响当前操作。

---

### 技巧 3：查看标签

如果忘记按钮功能，展开菜单后每个按钮左侧都有清晰的文字标签。

## ❓ 常见问题

### Q1：为什么点击 FAB 没有反应？

**A**：请确保：
- 应用已完全加载
- 没有其他弹窗覆盖
- 点击的是右下角的圆形按钮

---

### Q2：菜单展开后如何关闭？

**A**：三种方式：
1. 再次点击主按钮（× 图标）
2. 点击屏幕上的背景遮罩
3. 选择任一菜单项

---

### Q3：能否自定义菜单项？

**A**：目前版本（v1.0.0）不支持自定义，未来版本（v1.1）将支持。

---

### Q4：为什么点击"添加 Agent"后没有直接进入添加表单？

**A**：为了保持一致性，点击后会先跳转到 Agent 列表页面，在那里可以看到现有的 Agent，然后再选择添加新的 Agent。

---

### Q5：动画可以更快或更慢吗？

**A**：当前动画时长固定为 300ms（Material Design 推荐值），未来版本可能支持调整。

- `_toggleFabMenu()` - 切换菜单开关
- `_buildFabMenuItem()` - 构建菜单项

---

### 扩展新菜单项

如果需要添加新的菜单项，参考以下代码：

```dart
AnimatedBuilder(
  animation: _translateButton,
  builder: (context, child) {
    return Transform.translate(
      offset: Offset(0, -(_translateButton.value * 210)), // 位置：140 + 70
      child: Opacity(
        opacity: _translateButton.value,
        child: _buildFabMenuItem(
          icon: Icons.search,           // 图标
          label: '搜索',                 // 标签
          backgroundColor: Colors.green, // 颜色
          onTap: () {
            _toggleFabMenu();
            // 导航到搜索页面
          },
        ),
      ),
    );
  },
),
```
