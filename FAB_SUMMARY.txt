  • 添加 SingleTickerProviderStateMixin
  • 新增动画控制器

  1. AnimationController        - 控制动画

🎯 交互流程
══════════════════════════════════════════════════════════════════════════════

流程 1: 展开菜单
  1. 用户点击 FAB 按钮
  2. 主按钮图标变化: + → ×
  3. 菜单项从下向上展开（300ms）
  4. 背景遮罩显示

流程 2: 关闭菜单
  方式 A: 点击主按钮（× 图标）
  方式 B: 点击背景遮罩
  方式 C: 选择菜单项后自动关闭

流程 3: 添加 Agent
  1. 点击 FAB → 展开菜单
  2. 点击 "添加 Agent" （蓝色按钮）
  3. 跳转到 Agent 列表页面
  4. 点击列表页的 + 按钮
  5. 选择 Agent 类型并创建

流程 4: 创建群组
  1. 点击 FAB → 展开菜单
  2. 点击 "创建群组" （紫色按钮）
  3. 跳转到创建群组页面
  4. 填写表单并选择成员
  5. 点击 "创建" 完成
