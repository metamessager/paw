# 实时聊天功能演示

## 📱 演示视频说明

本指南展示了 AI Agent Hub 的实时聊天功能。

## 🎬 功能演示场景

### 场景 1: 添加 Agent

**步骤**:
1. 启动应用
2. 点击右上角「+」按钮
3. 填写 Agent 信息：
   ```
   名称: GPT 助手
   类型: Remote Agent (A2A 协议)
   端点: https://api.openai.com/v1/agent
   Token: sk-xxxxxxxxxxxxxxxx
   ```
4. 点击「保存」

**预期结果**: Agent 添加成功，显示在 Agent 列表中

### 场景 2: 开始对话

**步骤**:
1. 在 Agent 列表中找到"GPT 助手"
2. 点击 Agent 卡片进入详情页
3. 点击「开始对话」按钮

**预期结果**: 进入聊天界面，显示历史消息（如果有）

### 场景 3: 发送消息

**步骤**:
1. 在底部输入框中输入: "你好，请介绍一下自己"
2. 点击发送按钮（➤）
3. 观察界面变化

**预期结果**:
- 消息显示在右侧（蓝色气泡）
- 显示"打字中"状态
- Agent 回复显示在左侧（灰色气泡）
- 自动滚动到最新消息

### 场景 4: 多轮对话

**步骤**:
1. 继续发送: "你能帮我写一段 Python 代码吗？"
2. 等待回复
3. 继续发送: "解释一下这段代码"

**预期结果**:
- 所有消息按时间顺序显示
- 自动区分发送方和接收方
- 消息自动保存，重新打开时仍可见

### 场景 5: 错误处理

**步骤**:
1. 添加一个不存在的 Agent（端点错误）
2. 尝试发送消息

**预期结果**:
- 显示错误提示
- 用户消息仍然保存
- 显示系统错误消息

### 场景 6: 加载历史消息

**步骤**:
1. 关闭聊天界面
2. 重新进入 Agent 详情页
3. 点击「开始对话」

**预期结果**:
- 自动加载历史消息
- 消息按时间顺序显示
- 滚动到最新位置

## 🔧 技术细节

### 消息发送流程

1. 用户输入消息
2. ChatScreen 调用 ChatService.sendMessageToAgent()
3. ChatService 保存用户消息到数据库
4. ChatService 通过 A2A 协议发送消息到 Agent
5. Agent 处理并返回响应
6. ChatService 解析响应并保存到数据库
7. ChatScreen 更新界面显示

### 数据存储

- **消息表**: 存储所有对话消息
- **频道表**: 管理用户-Agent 对话频道
- **Agent 表**: 存储所有注册的 Agent

### 协议支持

- **A2A 协议**: Agent-to-Agent Protocol（推荐）
- **HTTP 协议**: 通用 HTTP/HTTPS 请求

## 📊 性能指标

### 响应时间

- 本地消息保存: < 50ms
- 历史消息加载: < 100ms（100 条消息）
- Agent 响应: 取决于 Agent 性能和网络

### 内存占用

- 基础应用: ~50MB
- 加载 100 条消息: ~10MB

### 存储占用

- 每条消息: ~1KB
- 1000 条消息: ~1MB

## ❓ 常见问题

### Q: 消息发送后没有回复？

**A**: 检查以下几点：
1. Agent 端点地址是否正确
2. Token 是否有效
3. 网络连接是否正常
4. Agent 是否在线

### Q: 消息历史显示不完整？

**A**: 默认加载最近 100 条消息，可以修改 limit 参数加载更多。

### Q: 如何删除聊天记录？

**A**: 在 Agent 详情页面，点击「删除聊天」按钮即可删除所有消息。

### Q: 支持多 Agent 同时对话吗？

**A**: 支持。你可以与多个 Agent 分别对话，每个对话独立保存。

## 🎯 使用建议

### 最佳实践

1. **清晰描述问题**: 详细描述你的需求，Agent 能更好地理解
2. **分段提问**: 复杂问题可以分成多个小问题
3. **提供上下文**: 引用之前的对话内容，帮助 Agent 理解
4. **及时反馈**: 对 Agent 的回复提供反馈，帮助优化

### 提示技巧

1. **具体化**: 不要说"写代码"，要说"写一个 Python 函数计算斐波那契数列"
2. **指定格式**: "用 Markdown 格式返回"
3. **要求解释**: "请解释这段代码的工作原理"
4. **提供示例**: "像这样处理: 示例代码"
