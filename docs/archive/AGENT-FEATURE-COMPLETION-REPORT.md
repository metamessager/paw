## 🎯 使用流程

### 查看 Agent 列表
1. 进入主页
2. 点击 "Agent 管理" 卡片
3. 查看所有 Agent 列表
4. 下拉刷新更新列表

### 添加新 Agent
1. 在 Agent 列表页点击 "➕ 添加 Agent"
2. 填写 Agent 信息
   - 名称（必填）
   - 类型（必填）
   - 状态（默认在线）
   - Avatar URL（可选）
3. 点击 "创建" 按钮
4. 自动返回列表并刷新

### 编辑 Agent
1. 在列表中点击 Agent 卡片
2. 查看详情后点击编辑按钮
3. 修改信息
4. 保存或取消

### 删除 Agent
1. 在列表中点击 Agent 卡片右侧的 ⋮ 按钮
2. 选择 "删除"
3. 确认删除对话框
4. 自动刷新列表
